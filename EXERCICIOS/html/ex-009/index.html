Lista de definição

Lista de definição básicamente é uma lista que pode ser usada em diversas ocasiões, uma delas seria em definição do que é ou oque sognifica uma palavra ou um item em questão, exemplo:

HTML
É uma linguafgem de marcação de hyper texto que nos da a possibilidade de adicionar link na pagina en =tre outras coisas.
CSS
É uma linguagem de stylo para aplicar formas, cores e efeitos ao conteúdo escrido em HTML.
Java Script
É uma linguagem de programação para aplicar a lógica e fazer interação com banco de dados nos sites escritos em HTML e CSS.
