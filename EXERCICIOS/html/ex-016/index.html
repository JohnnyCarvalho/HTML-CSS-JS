Aplicando estilos em CSS

Um modo de inserir estilo CSS em HTML mais recomendável seria criando uma pasta fora chamada stile.css e chamá-la dentro do HTML usando a tag <link rel="stylesheet" href="style.css">.

Nesse exemplo em questão nós utilizamos essa tecnica para estilizar e dar cores, tando ao fundo dapágina, quando as letras atacando a tag <p> e a tag <h1> separadamente.
