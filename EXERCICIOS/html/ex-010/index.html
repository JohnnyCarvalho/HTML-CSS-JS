Links externos

É possível acessar links externos em um site HTML, basta usarmos o comando <a href=""></a>, esse comando é conhecido como âncora. O comando em questão funciona assim, vamos criar um link para o YouTube aqui. Existe um porém aqui.
Se deixarmos o link dessa forma <a href="https://youtube.com">YouTube</a>, ao clicarmos no link, ele vai nos levar ao site desejado (nesse caso o YouTube). Porém, nota que ele não vai abriir uma nova guia, ou seja, ele vai subtituir a guia do seu site para abrir a o link, ou seja, vovê vai estar em palavras simples, expulsando o possível cliente ou visualizador do seu site, má, existe uma forma de consertar isto, vamos usar o comando <a href="https://youtube.com" target="_blank" rel="external">YouTube</a>, vejamos um exemplo desse comando aplicado nesse novo link para o YouTube, é importante ressaltar que, essa função só é recomendada em links externos, ou seja, quando você vai levar o usuário para outro site, ou blog, ou documento etc... porém, quando o usuário for acessar um link que conste dentro do próprio site, aí nesse caso em questão não utilizamos esse comenado target="_blank" rel="external", é muito importante ficar claro isso.
